中国第一程序员：雷军是我小弟！国内最壕的8大程序员

思诚开发者沙龙 2017-04-26 17:29

思诚科技的思诚New学堂有一批批程序员走上工作岗位，工资也都比较可观。都说程序员工资高，那为什么程序员的工资高呢？因为技术好，人才少啊！

但是现在互联网IT公司到处都是，行业的从业者来看，人数一点儿也不少，而且程序员这个行业的薪资差距可谓云泥之别，有人整天顶着黑眼圈加班，经常看到程序员猝死的新闻，有人靠自己的技术走上人生的巅峰，成为科技大佬，建立自己的商业帝国。

现在我们就来看看中国最壕的十大程序员，不仅仅是工资高，而且是壕！

01：网易老大丁磊

不会养猪的游戏玩家不是好程序员。

网易公司创始人，现担任网易公司董事局主席兼首席执行官。1997年5月创立网易公司，与张朝阳、王志东并称为网络三剑客。2016年2月24日，丁磊 财富630亿元新进《2016胡润全球富豪榜》大陆前十名，位列第十，全球第121位，比2015年上升30位。

02：QQ软件开发者马化腾

抄袭是成功的最快途径，谁说我抄袭我开除谁

腾讯公司主要创办人之一，现担任腾讯公司控股董事会主席兼首席执行官。曾在深圳大学主修计算机及应用，1993年取得深大理科学士学位。《2016胡润全球富豪榜》发布，马化腾以财富1240亿元，排名中国富豪第四。

03：小米科技创始人雷军

哥当年没有投马云，是因为马总比我都能吹

小米科技创始人、董事长兼首席执行官；金山软件公司董事长，2014年2月，雷军首次以280亿元财富进入“胡润全球富豪榜”，跃居大中华区第57名，全球排名第339位。2016年以98亿美元居福布斯第二十二位。

04：巨人ceo史玉柱

哥的第一桶金不是卖脑白金而是开发软件赚得的

2016胡润IT富豪榜发布，史玉柱家族以300亿元排名第11。10月27日，2016福布斯中国富豪榜公布，史玉柱排名第46位

05：百度创始人李彦宏

百度代表网民的立场，我们和网民永远站在一起

2000年1月，李彦宏创建了百度。经过十多年的发展，百度已经发展成为全球第二大独立搜索引擎和最大的中文搜索引擎，也成为中国最具价值的品牌之一。《2016胡润全球富豪榜》发布，李彦宏、马东敏夫妇以财富825亿元，排名中国富豪第7名。

06：微信之父张小龙

我开发了微信，但是微商跟我没有关系

Foxmail创始人，微信创始人，腾讯公司高级副总裁。曾开发国产电子邮件客户端--Foxmail，加盟腾讯公司后开发腾讯微信，被誉为"微信之父"，被《华尔街日报》评为"2012中国创新人物"。

07：360公司ceo周鸿祎

流氓不可怕就怕流氓有文化

创办和领导下的360公司是全球第二、中国最大的互联网安全企业，并于2011年3月30日在美国纽交所上市，2016年7月15日，完成私有化。

08：中国第一程序员求伯君

雷军是我小弟。

毕业于中国人民解放军国防科技大学，有"中国第一程序员"之称。

1994年，在珠海创立珠海金山电脑公司，任董事长兼总经理。2000年底，担任金山公司执行董事及董事会主席。2011年07月06日，正式公布其退休计划，副董事长雷军接任金山公司董事长。

这些可都是程序员出身的大佬，就问你壕不壕！

其实，软件行业之所以高薪，是因为这个行业技术更新发展快，在这个行业，技术几乎每五年就翻新一轮，这带来一个结果，就是在这个行业每一位从业者，其实，都有一个统一的别称：学生！学无止境嘛。
